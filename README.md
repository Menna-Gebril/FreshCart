# E-Commerce Website

A responsive E-Commerce web application built with React.


The project allows users to browse products, view product details, and manage their shopping experience through a modern and user-friendly interface.


## Features
-Responsive design for all screen sizes
-Product listing and details page
-Categories filtering
-Shopping cart functionality
-API integration
-Dynamic routing using React Router
-Clean and modern UI


## Technologies Used
-React.js
-Axios
-React Router
-Tailwind CSS
-Vite
